# MindWell — University Mental Health Support Platform

A complete, production-ready website for university mental health support services.

## Features

- **4 Service Cards** — Individual Counseling, Group Therapy, Mindfulness & Meditation, Crisis Support. Clicking any card smoothly scrolls to its corresponding detail section.
- **Detail Sections** — Each service has a dedicated section with full information, feature lists, and call-to-action buttons.
- **Messenger-like Chat** — Contact list panel, conversation panel, message bubbles, text input with send button, and an auto-reply bot for demo purposes.
- **Responsive Design** — Works on desktop, tablet, and mobile.
- **Smooth Animations** — Scroll-triggered reveals, hover effects, floating background shapes.
- **Modern UI/UX** — Clean, calming design with a warm color palette.

## Usage

Open `index.html` in any modern browser. No build step or server required.

## Tech Stack

- Vanilla HTML5 / CSS3 / JavaScript (ES5+)
- Google Fonts (Inter)
- Font Awesome 6 (icons)
- No frameworks, no build tools — runs immediately.
